📦 ระบบนับสต็อกสินค้า + Google Sheets

บันทึกในมือถือ แสดงผลใน Google Sheets แบบ Real-time

🔴 ยังไม่เชื่อมต่อ Google Sheets
⚙️ การตั้งค่า Google Sheets

ขั้นตอนการเชื่อมต่อ:

  1. เปิด Google Sheets สร้างชีทใหม่
  2. ตั้งชื่อ Column A1=รหัสบาร์โค้ด, B1=ชื่อสินค้า, C1=จำนวน, D1=ราคาขาย, E1=เวลาบันทึก
  3. ไปที่ Extensions → Apps Script
  4. วาง Code ด้านล่างแล้ว Deploy เป็น Web App
  5. คัดลอก URL มาใส่ในช่องด้านล่าง
